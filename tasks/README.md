# Task Tracking (clwclw-monitor)

이 디렉토리는 `REQUIREMENTS.md`의 요구사항을 구현하기 위한 작업 추적용 문서들입니다.

## 규칙

- 파일명: `NNNN-짧은-제목.md` (예: `0001-bootstrap-coordinator.md`)
- 각 태스크 문서는 아래를 포함합니다:
  - `Status:` (Todo / In Progress / Blocked / Done)
  - `Goal:` (한 줄)
  - `Acceptance Criteria:` (체크리스트)
  - `Notes / References:` (관련 파일 경로, API, 결정사항)
- 완료/진행 상태는 이 파일들에서 계속 업데이트합니다(“작업 추적” 목적).

## 인덱스

전체 태스크 목록은 `tasks/INDEX.md`를 기준으로 봅니다.
