FEATURES

Agent 관리의
모든 것

분산된 Claude Code 세션을 중앙에서 조율합니다. 작업 생성부터 완료까지, 하나의 화면에서.

📊

Real-time Dashboard

모든 에이전트의 상태, 작업 진행률, 이벤트 타임라인을 실시간으로 확인하세요.

⚡

FIFO Task Queue

채널 기반 작업 큐로 자동 분배. 원자적 task claim으로 중복 없는 안전한 처리.

🔗

Chain Execution

여러 작업을 순차 실행 체인으로 묶어 복잡한 워크플로우를 자동화하세요.

💻

Remote Control

tmux를 통한 원격 명령 주입. 대시보드에서 직접 Claude Code 세션을 제어합니다.

🔔

Multi-Channel Alerts

Telegram, Email, LINE, Desktop 알림. Claude가 멈추면 즉시 알려드립니다.

🤖

Multi-Session

여러 Claude Code 인스턴스를 동시에 운영. 각 세션이 독립적으로 작업을 처리합니다.

HOW IT WORKS

세 단계로 시작하세요

01

Coordinator 시작

Go 서버를 실행하면 대시보드와 API가 즉시 활성화됩니다. Docker로 한 줄이면 충분합니다.

02

Agent 연결

각 Claude Code 세션에서 Agent를 실행하면 자동으로 Coordinator에 등록됩니다.

03

작업 관리

대시보드에서 채널과 작업을 생성하면 Agent가 자동으로 claim하고 실행합니다.

SECURITY

보안을 최우선으로

로컬 환경의 민감한 정보를 다루기에, 보안은 타협 없이.

JWT Authentication

사용자 인증은 JWT 토큰 기반. 24시간 만료, 서버 상태 없이 안전한 세션 관리.

Password Policy

대문자, 특수문자 필수 포함. 최소 6자 이상. bcrypt 해싱으로 안전하게 저장.

API Token Guard

Agent 통신은 Bearer 토큰으로 보호. 무단 접근을 원천 차단합니다.
